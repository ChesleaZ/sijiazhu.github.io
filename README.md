# Sijia Zhu Personal Website

Personal portfolio built with Gatsby, React, and styled-components.

This site is adapted from Brittany Chiang's open-source v4 portfolio template. Please keep attribution if you publish it publicly.

## Develop

```bash
yarn
yarn develop
```

## Customize

See `CUSTOMIZE.md` for the main files to edit.

## Deploy

This repository includes a GitHub Actions workflow that builds Gatsby and deploys the generated `public` folder to GitHub Pages whenever `main` is updated.
